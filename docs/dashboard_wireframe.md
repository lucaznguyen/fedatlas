# Dashboard Wireframe

## Global Layout

The dashboard uses a hero header, a sticky left filter rail, and a main tabbed workspace.

Global filters:

- Year range
- Topic group
- Venue quality
- Code availability
- Minimum citations
- Minimum GitHub stars
- Top N
- Search

## Tabs

1. Overview: KPIs, growth chart, topic heatmap, top topics, top countries, top venues, insight cards.
2. Global Collaboration: world map, bridge-country ranking, country-pair collaboration table.
3. Topic Evolution: topic-year heatmap, stacked trend, rising topic table, topic detail table.
4. Network Explorer: heterogeneous `visNetwork` graph with node and edge filters.
5. Research-to-Code Gap: citation/code scatter, topic code adoption, high-citation no-code table, high-code impact table.
6. Repositories & Contributors: repository leaderboard, language distribution, activity chart, contributor table.
7. Venues & Quality: quality summary, citation/code adoption by quality, Unknown venue review.
8. Data & Methods: source notes, manifests, data quality summary, queries, export locations.
